# listr

自作の難解プログラミング言語 listr のインタプリタ `listr.hs`

## 言語の概要

データはすべてリストであり、文字列は各文字の文字コードの長さを持つリストのリストとして表される。

"ABC" ↔ [[ [] ×65 ], [ [] ×66 ], [ [] ×67 ]]

listr プログラムは入力文字列を出力文字列に書き換える命令として、以下を組み合わせて作られる。

|命令|内容|備考|
|:---:|---|---|
|`0`|(head) 先頭を取り出す: [_a_, ... ] → _a_ ; [] → []||
|`1`|(tail) 先頭を捨てる: [_a_, ... ] → [ ... ] ; [] → []||
|_x_`.`_y_|(cons) 命令 _x_ を実行した結果を、命令 _y_ を実行した結果の先頭に追加|右結合: _x_`.`_y_`.`_z_ := _x_`.(`_y_`.`_z_`)`|
|_x_`\|`_y_|(while-do) 命令 _x_ を実行した結果が [] になるまで命令 _y_ を繰り返し実行|右結合、`.`と同じ優先順位|
|`(`_x_`)`|グループ化||
|_xy_|命令 _x_ を実行してから命令 _y_ を実行||
|`#`...|コメント (改行まで)||

### チューリング完全性の証明

brainfuck (チューリング完全性が知られている) で書かれたプログラムは以下の手順で listr のプログラムに変換される。

1. 以下の置き換えを行う

  - `>` → `(1100.(0.10).1101.111)`
  - `<` → `(100.101.(0.110).111)`
  - `+` → `(0(0.).1)`
  - `-` → `(01.1)`
  - `.` → `(0.10.110.1110.0.1111)`
  - `,` → `(11100.10.110.11101.1111)`
  - `[` → `(0|`
  - `]` → `)`

2. 冒頭に `(.|0)(1.1.1.0.1)`、末尾に `1111(.|0)(0|01.00.1)1` を追加する

## 実行

```bash
runghc listr.hs program.listr < input
```

または

```bash
ghc -O2 listr.hs
./listr program.listr < input
```
